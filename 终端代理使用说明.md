# 终端代理使用说明

## 1）仅让当前终端命令走代理（最常用）

在终端执行（将端口改为你的代理端口）：

```bash
export http_proxy="http://127.0.0.1:7890"
export https_proxy="http://127.0.0.1:7890"
export all_proxy="socks5://127.0.0.1:7890"
```

### 验证是否生效

```bash
curl ipinfo.io
```

> 如果出现 `429 Rate limit`，说明接口被限流，不代表代理失败。可改用：
>
> ```bash
> curl -s https://api.ipify.org
> ```

### 取消代理

```bash
unset http_proxy https_proxy all_proxy
```

---

## 2）每次打开终端自动生效（zsh）

把上面的 `export` 写入 `~/.zshrc`，然后执行：

```bash
source ~/.zshrc
```
